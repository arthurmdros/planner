# 📚 Manual de Uso - Planner Digital Premium

## 🎯 Bem-vindo!

Este manual completo irá ajudá-lo a aproveitar ao máximo seu Planner Digital Premium para Concurso Público. O planner foi projetado para ser intuitivo, poderoso e altamente funcional.

---

## 🚀 Início Rápido

### 1. Abrindo o Planner
- Abra o arquivo `planner.html` em seu navegador moderno (Chrome, Firefox, Safari, Edge)
- O planner carregará automaticamente com a primeira página visível
- Todos os seus dados serão salvos automaticamente no navegador

### 2. Primeiros Passos
- **Navegação**: Use o menu lateral direito para alternar entre páginas
- **Edição**: Clique em qualquer campo cinza para começar a digitar
- **Salvamento**: Todos os dados são salvos automaticamente

---

## 📱 Navegação e Interface

### Menu Lateral
O menu lateral direito contém 8 abas de navegação:

1. **Semana** - Cronograma semanal de estudos
2. **Progresso** - Acompanhamento de questões resolvidas
3. **Revisões** - Sistema de revisão inteligente
4. **Controle** - Controle semanal de estudos
5. **Simulados** - Registro de simulados realizados
6. **Erros** - Caderno de erros para análise
7. **Desempenho** - Gráficos por disciplina
8. **Meta** - Definição de objetivos e motivação

**Como navegar:**
- Clique em qualquer aba para ir diretamente à página correspondente
- A aba ativa ficará destacada em azul
- Sua última página visitada será restaurada ao reabrir o planner

---

## ✏️ Edição e Campos

### Campos Editáveis
Todos os campos cinza pontilhados são totalmente editáveis:

**Tipos de campos:**
- **Campos de texto**: Para digitar informações
- **Checkboxes**: Para marcar conclusão de tarefas
- **Trackers**: Para controle visual de progresso

**Como editar:**
1. Clique em qualquer campo cinza
2. Digite suas informações
3. Pressione Enter ou clique fora para salvar
4. O campo ficará azul quando preenchido

**Características dos campos:**
- **Salvamento automático**: Tudo é salvo instantaneamente
- **Visual feedback**: Campos mudam de cor quando preenchidos
- **Foco visual**: Campo destacado ao clicar
- **Restauração**: Dados são recuperados ao reabrir

- Tabela completa com horários de segunda a sexta
- Disciplinas organizadas por cores
- Espaços para marcar conclusão

**Como usar:**
- Clique nos checkboxes para marcar tarefas concluídas

- Organize seu dia conforme os horários pré-definidos

**Horários disponíveis:**

- 09:15 – 10:30 (Bloco 2)
- 10:30 – 10:45 (Pausa)
- 10:45 – 12:00 (Bloco 3)
- 12:00 – 13:15 (Almoço)
- 13:15 – 14:30 (Bloco 4)
- 14:30 – 15:30 (Treino físico)
- 15:45 – 17:00 (Bloco 5)
- 17:00 – 18:15 (Bloco 6)

### 2. Progresso Semanal de Questões (Página 2)

**O que é:**
- Sistema de acompanhamento de meta de 600 questões/semana
- Registro diário de questões resolvidas
- Barra de progresso visual
- Tracker com quadrados de 50 questões

**Como usar:**
- Preencha o número de questões resolvidas diariamente
- Clique nos quadrados do tracker para marcar 50 questões
- A barra de progresso atualiza automaticamente
- O total semanal é calculado automaticamente

**Tracker Visual:**
- Cada quadrado representa 50 questões
- Clique para marcar como concluído
- Cor verde indica progresso
- Total: 12 quadrados = 600 questões

### 3. Sistema de Revisão Inteligente (Página 3)

**O que é:**
- Método de repetição espaçada
- Controle de revisões em 24h, 7 dias e 30 dias
- Base científica para memorização

**Como usar:**
1. **Assunto estudado**: Digite o conteúdo revisado
2. **Data de estudo**: Registre quando estudou
3. **Revisões**: Marque os checkboxes conforme faz as revisões
4. **Espaço para 15 assuntos diferentes**

**Ciclo de revisão:**
- **24 horas**: Primeira revisão para fixação inicial
- **7 dias**: Segunda revisão para consolidação
- **30 dias**: Terceira revisão para memória de longo prazo

### 4. Controle Semanal de Estudos (Página 4)

**O que é:**
- Registro detalhado de atividades diárias
- Controle de horas estudadas
- Contagem de questões resolvidas
- Campo para observações importantes

**Como usar:**
- Preencha diariamente as informações
- Use o campo "Observações" para notas importantes
- O resumo semanal ajuda a identificar padrões

**Campos disponíveis:**
- **Dia**: Dias da semana (segunda a domingo)
- **Horas estudadas**: Tempo total de estudo
- **Questões resolvidas**: Número de questões do dia
- **Disciplinas estudadas**: Matérias abordadas
- **Observações**: Notas e insights

### 5. Controle de Simulados (Página 5)

**O que é:**
- Registro de até 12 simulados
- Controle de pontuação e acertos
- Análise de erros frequentes
- Estatísticas automáticas

**Como usar:**
- Registre cada simulado realizado
- Anote pontuação e número de acertos
- Descreva os principais erros para estudo focado
- Use as estatísticas para acompanhar evolução

**Estatísticas automáticas:**
- Média de pontuação
- Maior pontuação alcançada
- Total de simulados realizados
- Taxa de acerto média

### 6. Caderno de Erros (Página 6)

**O que é:**
- Espaço para análise detalhada de erros
- Sistema de aprendizado com falhas
- Controle de revisão de erros corrigidos

**Como usar:**
1. **Disciplina**: Matéria da questão errada
2. **Questão errada**: Número ou descrição da questão
3. **Motivo do erro**: Por que errou (falta de conhecimento, distração, etc.)
4. **Correção**: Explicação correta
5. **Revisado**: Marque quando revisar o erro

**Dica:** Use este caderno ativamente para não repetir os mesmos erros.

### 7. Desempenho por Disciplina (Página 7)

**O que é:**
- Tabela com estatísticas por matéria
- Gráfico visual de performance
- Identificação de pontos fortes e fracos

**Como usar:**
- Preencha os dados após cada sessão de estudo
- O gráfico atualiza automaticamente
- Use para direcionar seus estudos

**Campos da tabela:**
- **Disciplina**: Nome da matéria com cor identificadora
- **Questões feitas**: Total de questões resolvidas
- **Questões certas**: Número de acertos
- **Taxa de acerto**: Percentual de sucesso

**Gráfico visual:**
- Barras coloridas por disciplina
- Altura representa performance
- Interativo com efeito hover

### 8. Meta de Aprovação (Página 8)

**O que é:**
- Página motivacional e de planejamento
- Definição clara de objetivos
- Plano de estudo estruturado

**Como usar:**
- **Cargo desejado**: Seja específico sobre seu objetivo
- **Data da prova**: Defina uma data clara
- **Motivo para aprovação**: Seus motivos pessoais
- **Plano de estudo**: Estratégia detalhada até a prova

**Frase motivacional:**
Use a citação inspiradora para manter-se focado e motivado.

## 💾 Sistema de Persistência de Dados

### Como Funciona
- **LocalStorage**: Todos os dados são salvos no navegador
- **Automático**: Salvamento instantâneo após qualquer alteração
- **Persistente**: Dados permanecem mesmo após fechar o navegador
- **Seguro**: Apenas você tem acesso aos seus dados

### Backup e Restauração

**Exportar Dados:**
1. Clique no botão "Exportar Dados" (canto superior esquerdo)
2. O arquivo .json será baixado automaticamente
3. Salve em local seguro como backup

**Importar Dados:**
1. Clique em "Importar Dados"
2. Selecione o arquivo .json salvo anteriormente
3. Os dados serão restaurados automaticamente

**Limpar Dados:**
- Clique em "Limpar Dados" para apagar tudo
- Use com cuidado - esta ação não pode ser desfeita
- Ideal para recomeçar ou transferir para outra pessoa

## 🔧 Solução de Problemas

### Problemas Comuns

**Dados não estão salvando:**
- Verifique se o localStorage está habilitado
- Tente usar um navegador diferente
- Limpe o cache e recarregue

**Campos não editáveis:**
- Atualize a página
- Verifique se está usando um navegador compatível
- Tente clicar em outro campo primeiro

**Navegação não funciona:**
- Recarregue a página
- Verifique o console do navegador para erros
- Certifique-se de que o JavaScript está habilitado

**Perda de dados:**
- Use regularmente a função exportar
- Mantenha backups em locais seguros
- Importe dados se necessário

### Contato e Suporte
Caso encontre problemas não listados aqui:
1. Verifique se está usando a versão mais recente
2. Tente em outro navegador
3. Exporte seus dados antes de tentar soluções drásticas
