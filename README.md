## Planner Digital Premium para Concurso Público

Um planner digital interativo e profissional otimizado para uso em tablets com caneta (Apple Pencil, S Pen, etc.).

## 🎯 Características

- **8 páginas organizadas** para controle completo dos estudos
- **Design premium minimalista** com cores suaves e layout limpo
- **Otimizado para tablet** com espaços amplos para escrita manual
- **Abas laterais interativas** para navegação rápida
- **Sistema de revisão espaçada** para fixação do conteúdo
- **Cores codificadas por disciplina** para organização visual

## 📚 Páginas Incluídas

### 1. Cronograma Semanal
- Tabela semanal com horários definidos (08:00-18:15)
- Disciplinas organizadas por cores
- Caixas de check para marcar conclusão
- Pausas e tempo para almoço/treino físico

### 2. Progresso Semanal de Questões
- Meta de 600 questões semanais
- Registro diário de questões resolvidas
- Barra de progresso visual
- Tracker visual com quadrados de 50 questões

### 3. Sistema de Revisão Inteligente
- Controle de revisões em 24h, 7 dias e 30 dias
- Baseado em repetição espaçada
- Espaço para 15 assuntos diferentes

### 4. Controle Semanal de Estudos
- Registro de horas estudadas por dia
- Contagem de questões resolvidas
- Campo para observações
- Resumo semanal automático

### 5. Controle de Simulados
- Espaço para 12 simulados
- Registro de pontuação e acertos
- Campo para principais erros
- Estatísticas automáticas

### 6. Caderno de Erros
- Espaço para 20 erros registrados
- Análise de motivo e correção
- Check de revisão
- Organização por disciplina

### 7. Desempenho por Disciplina
- Tabela com estatísticas por matéria
- Gráfico visual de performance
- Taxa de acerto automática
- Cores codificadas

### 8. Meta de Aprovação
- Espaço para cargo desejado
- Data da prova
- Motivação pessoal
- Plano de estudo detalhado

## 📱 Como Usar

### Opção 1: HTML Interativo (Recomendado)
1. Abra o arquivo `planner.html` no seu navegador
2. Use as funções interativas (checks, progresso, etc.)
3. Imprima como PDF quando necessário
4. Use em tablets com apps de anotação

### Opção 2: PDF Direto
1. Abra `planner.html` no navegador
2. Pressione `Ctrl+P` (Windows) ou `Cmd+P` (Mac)
3. Salve como PDF
4. Importe para GoodNotes, Notability, Samsung Notes

### Opção 3: Python (Avançado)
```bash
pip install reportlab Pillow matplotlib
python generate_planner.py
```
